### Jon Pretty

Rapture and Typelevel Scala developer  

Experienced Scala developer, consultant and evangelist. Jon has been involved with Scala for the last eight years and has overseen numerous successful Scala deployments.

#### Towards Dependently-typed Checked Exceptions in Scala

Jon will start by demonstrating some of the new features Rapture which provide better compile-time and runtime safety for working with JSON or XML in Scala, then show how combining these ideas with typed multimaps can provide dependently-typed, checked exceptions in Scala.
